# caveman

Talk like smart caveman. Same brain, fewer tokens.

## What it does

Compresses model responses by dropping filler, pleasantries, hedging, and
optional grammar while preserving technical accuracy, code blocks, paths, error
strings, flags, URLs, and exact symbols.

## How to invoke

```text
/caveman
/caveman lite
/caveman full
/caveman ultra
/caveman wenyan-lite
/caveman wenyan-full
/caveman wenyan-ultra
stop caveman
normal mode
```
